================================================================================
MODÈLE DJANGO - JOBOFFER
================================================================================

Champs:
  - title: Titre du poste
  - slug: Slug unique
  - description: Description du poste
  - requirements: Exigences
  - benefits: Avantages
  - department: Département
  - location: Localisation
  - contract_type: Type de contrat
  - salary_min: Salaire minimum
  - salary_max: Salaire maximum
  - currency: Devise (USD, EUR, etc.)
  - status: Statut (ouvert, fermé, archivé)
  - deadline: Date limite de candidature
  - is_featured: En vedette
  - order: Ordre d'affichage

INSTRUCTIONS DE DÉMARRAGE
================================================================================

BACKEND (Terminal 1):
  cd mwolo-energy-systems
  venv\Scripts\activate  (Windows) ou source venv/bin/activate (Linux/Mac)
  python manage.py makemigrations cms
  python manage.py migrate
  python manage.py populate_data
  python manage.py runserver

FRONTEND (Terminal 2):
  cd mwolo-energy-systems/frontend
  npm run dev

PUIS OUVRIR:
  http://localhost:3000 (Frontend)
  http://localhost:8000 (Backend)
  http://localhost:8000/mwoloboss/ (Admin Django)
